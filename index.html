What is web Scraping?

Web Scraping is a technique in which data is extracted from websites by parsing the HTML code of the websites. By running a short piece of code, we can automatically pull specific information from websites. It is very helpful in gathering major sources of data more effeciently and error-prone than manually viewing the data.

How can I Web Scrape?

There are many ways to start Web Scraping with several languages. The most beginner-friendly for Web Scraping is Python, so let's create a simple web scraper and try using it. First, head over to replit.com and create an account if you don't already have one. Next, create a new Python repl on replit.com (see the image on the left for how this should look). Using this repl, let's scrape quotes from the website: quotes.toscrape.com

Now what?

Copy and paste the following code into main.py:

import requests
from bs4 import BeautifulSoup

# Define the URL of the webpage you want to scrape
 url = ''

# Send an HTTP GET request to the URL
 response = requests.get(url)

# Check if the request was successful (status code 200)
 if response.status_code == ___:

# Parse the HTML content of the page with BeautifulSoup
 soup = BeautifulSoup(response.content, 'html.parser')

# Find the HTML elements that contain the quotes and their authors
 quote_elements = soup.find_all('span', class_='___')
author_elements = soup.find_all('small', class_='___')

# Loop through the found elements and extract the text (quotes and authors)
 for quote, author in zip(quote_elements, author_elements):
print(f'Quote: {quote.text}')
print(f'Author: {author.text}\n')
else:
print(f"Failed to retrieve the webpage. Status code: {response.status_code}")
